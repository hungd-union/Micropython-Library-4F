* This is a micropython library for LM393 Optical Speed Encoder. 

* For this code, you will need a:
- SLOT TYPE IR OPTOCOUPLER SPEED SENSOR MODULE LM393 3.3V-5V
- Coded Disc Encoder 20 Holds
- TT Hobby Gearmotor

* The library includes
- A construction for the Encoder objects (D0_Pin, Encoder PPR)
- A function to get the period of High Pulse
- A function to get instantaneous RPM
- A function to get average RPM
